## Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (React 19, TypeScript)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com)
- **Animations:** [Motion](https://motion.dev)
- **UI Components:** [Radix UI](https://www.radix-ui.com) (Toast, Tooltip, Label, Separator)
- **Icons:** [Lucide React](https://lucide.dev), [React Icons](https://react-icons.github.io/react-icons)
- **Contact Form:** [EmailJS](https://www.emailjs.com), [React Hook Form](https://react-hook-form.com)
- **Theming:** [next-themes](https://github.com/pacocoursey/next-themes) (dark mode by default)

## Getting Started

```bash
# Install dependencies
npm install

# Copy and configure environment variables
cp .env.local.example .env.local

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
